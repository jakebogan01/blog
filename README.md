# Basic Blog (Not Finished)

## Details
Not yet finished, basic blog. This project uses <a href="https://laravel.com" target="_blank">Laravel</a>.

## Installation
```
npm install
```

### install packages
```
composer install
```

### copy .env.example
```
cp .env.example .env
```

### generate key
```
php artisan key:generate
```

### Compiles
```
npm run dev
```

### create tables in databse
```
php artisan migrate
```

### create server
```
php artisan serve
```

Enjoy :palm_tree:
